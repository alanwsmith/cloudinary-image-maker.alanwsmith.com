Cloudinary URL Maker

Nothing fancy. Put some JSON in the box, click the button, and you'll get an image preview along with the correspoding production URL.


Notes

  • I mainly built this to make it easier to figure out text placement for open graph social images. It's super handy for that
  • The preivew image widht is hard coded right now. Production URLs are unaffected by this and will resize to whatever dimensions you spec
  • Cloudinary's api does a bunch of stuff. I'm sure this doesn't cover everything, but it'll get you pretty far. (Worst case, you can just drop an entire string in with whatever you need for a section.)

Usage

  • Both object keys and values require double quotes
  • Trailing commas don't work (e.g. something like this that you usually see on separate lines: `['a', 'b', 'c',]`
  • Each "seciton" in the "sections" array represents code that will go in betwee slashes in the final url
  • Each section is itself an array of item objects
  • Each item object has exactly one key and one value
  • The value of the item objects are always arrays of parameters
  • The item object parameters array always exists, but it can be empty
  • In sections, individual items are joined with commas
  • If there are parameters on an item, those are joined to their corresponding key with ":" characters.
  • Text is in the parameters is automatically URI encoded
